# Sprite Assets

Upload your sprite sheets from the Tiny RPG Character Asset pack here.

## Expected files

### characters/soldier/
- `Soldier-Idle.png` — idle animation frames
- `Soldier-Walk.png` — walk cycle frames
- `Soldier-Attack01.png` — attack animation
- `Soldier-Attack02.png` — attack animation
- `Soldier-Attack03.png` — attack animation
- `Soldier-Hurt.png` — hurt/hit reaction
- `Soldier-Death.png` — death animation

### characters/orc/
- `Orc-Idle.png` — idle animation frames
- `Orc-Walk.png` — walk cycle frames
- `Orc-Attack01.png` — attack animation
- `Orc-Attack02.png` — attack animation
- `Orc-Hurt.png` — hurt/hit reaction
- `Orc-Death.png` — death animation

### arrow/
- `Arrow.png` — projectile sprite

## Sprite sheet format
- Frame size: 100x100 pixels
- All sheets are horizontal strips (frames side by side)
